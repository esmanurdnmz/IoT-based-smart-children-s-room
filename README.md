# IoT-Based Smart Baby Room Monitoring System

An end-to-end IoT ecosystem for real-time tracking of baby room environments, developed under the TÜBİTAK 2209-A program.



## 🏗 System Overview
The system uses an **ESP32 Gateway** to collect BLE signals from **Xiaomi sensors**, transmits data via **AWS IoT Core**, and manages a **Raspberry Pi server** for data visualization and storage.

## 🌟 Highlights
* **Encrypted Communication:** Full TLS/SSL integration.
* **Historical Data:** MongoDB integration for trend analysis.
* **Scalable Design:** Can be extended with additional IoT sensors.
